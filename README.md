# 天枢金融核心平台（Tianshu Financial Core Platform）

天枢金融核心平台，是一个面向下一代金融机构建设的企业级核心系统群项目，目标是通过 AI Engineering、领域驱动设计（DDD）、微服务架构与自动化治理体系，探索从 0 到 1 构建现代银行核心能力平台的最佳实践。

本项目聚焦金融核心业务领域，覆盖但不限于：

* 交易系统（Transaction）
* 支付系统（Payment）
* 账务系统（Ledger）
* 清结算系统（Clearing & Settlement）
* 对账系统（Reconciliation）
* 风控系统（Risk Control）
* 用户中心 / 客户中心（Customer Center）
* 产品中心（Product Factory）
* 额度中心（Limit Management）
* 卡系统（Card Core）
* 信贷流程管理系统（Loan Workflow）
* 信贷核心系统（Loan Core）

## 项目愿景

打造一套具备高可用、高一致性、高扩展性、高治理能力的金融核心平台架构，并验证 AI 在复杂企业级系统研发中的真实生产力，包括：

* AI 驱动需求分析与规格设计（Spec-Driven Development）
* 多 Agent 协作研发流程治理
* 自动生成代码、测试与文档
* 面向金融场景的领域建模实践
* 核心系统群架构沉淀与复用

## 技术方向

* Java / Spring Boot / Spring Cloud
* PostgreSQL / MySQL / Redis / Kafka
* Kubernetes / Cloud Native
* DDD / CQRS / Event-Driven Architecture
* Hexagonal Architecture
* AI Agent Workflow / Spec-Kit / DevOps Automation

## 项目定位

这不是单一系统项目，而是一个持续演进的“金融核心操作系统”实践工程，用于沉淀未来银行级、支付级、信贷级系统建设方法论。

## 当前阶段

项目处于规划与验证阶段，首期将从账务系统（Ledger）作为核心切入点，逐步扩展至支付、信贷、风控等关键域。
